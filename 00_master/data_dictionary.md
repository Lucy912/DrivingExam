# Data Dictionary

## Validation Levels
- `verified`: Supported by statutory filing or official IR/webcast transcript.
- `partially_verified`: Supported by official secondary materials only.
- `unverified`: No official source support yet.

## Source Priority Levels
1. Annual report / 10-K / 20-F / statutory filing.
2. Quarterly or interim results material.
3. Investor presentation / CMD / webcast transcript.
4. Official press release or corporate webpage.
5. Third-party material (support only; cannot stand alone).

## Status Fields
- `not_started`
- `in_progress`
- `completed`
